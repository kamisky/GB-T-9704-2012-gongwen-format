# 公文要素编排速查表（GB/T 9704—2012）

## 三部分划分
```
┌─────────────────────────── 版心 (156×225mm) ───────────────────────────┐
│ 版头：首页红色分隔线(含)以上                                              │
│   份号 / 密级保密期限 / 紧急程度                                          │
│   发文机关标志(红,小标宋,上沿距版心顶35mm)                                │
│   发文字号(下空二行,居中) / 签发人(右空一字)                              │
│   ═══ 红色分隔线(发文字号下4mm,与版心等宽) ═══                            │
├─────────────────────────────────────────────────────────────────────────┤
│ 主体：红线下(不含) ~ 末页首条分隔线(不含)                                │
│   标题(2号小标宋,红线下空二行,居中,梯形/菱形)                            │
│   主送机关(标题下空一行,顶格,末尾全角冒号)                               │
│   正文(3号仿宋,左空二字回行顶格) + 层次(一/(一)/1./(1))                  │
│   附件说明(正文下空一行,左空二字)                                        │
│   发文机关署名 / 成文日期(右空四字) / 印章(红,居中下压)                  │
│   附注(成文日期下一行,左空二字,圆括号)                                    │
├─────────────────────────────────────────────────────────────────────────┤
│ 版记：末页首条分隔线(不含)~末条分隔线                                    │
│   ─── 首条粗线(0.35mm)                                                   │
│   抄送机关(4号仿宋,左右各空一字)                                         │
│   ─── 中间细线(0.25mm)                                                   │
│   印发机关(左空一字) / 印发日期(右空一字+"印发")                          │
│   ─── 末条粗线(0.35mm,与版心下边缘重合)                                  │
└─────────────────────────────────────────────────────────────────────────┘
页码：版心下边缘之下7mm，4号半角宋体，— N —，单页右空一字/双页左空一字
```

## 字体速查
| 用途 | 字体 | 字号 | 颜色 |
|---|---|---|---|
| 发文机关标志/纪要标志 | 方正小标宋简体(回退宋体) | 小初36pt | 红 |
| 标题 | 方正小标宋简体(回退宋体) | 二号22pt | 黑 |
| 正文/默认/署名/日期/附注/发文字号/主送/附件说明 | 仿宋 | 三号16pt | 黑 |
| 签发人姓名/层次第二层 | 楷体 | 三号16pt | 黑 |
| 密级/紧急/层次第一层/附件二字/出席二字 | 黑体 | 三号16pt | 黑 |
| 抄送/印发机关日期 | 仿宋 | 四号14pt | 黑 |
| 页码 | 宋体 | 四号14pt | 黑 |
| 份号 | 阿拉伯数字 | 三号16pt | 黑 |

## 层次序数
第一层 `一、` 黑体 → 第二层 `（一）` 楷体 → 第三层 `1.` 仿宋 → 第四层 `（1）` 仿宋

## 成文日期/印发日期数字
阿拉伯数字，年月日全标，月日不编虚位（1 不编 01）。例：`2026年7月16日`

## 发文字号
年份用六角括号 `〔〕`（非方括号[]或圆括号()）；顺序号不编虚位，后加"号"。例：`云办发〔2026〕12号`
