# 字体安装脚本

本项目已打包 **方正小标宋简体** 字体（`fonts/方正小标宋简体.ttf`），用于公文红头和标题。

## 快速安装

```bash
python scripts/install_font.py fonts/方正小标宋简体.ttf
```

安装后重启 Word/WPS，字体栏搜索"方正小标宋简体"确认可用。

## 字体说明

- **用途**：发文机关标志（红头）、标题（2号）
- **来源**：方正字库官方免费个人版
- **授权**：个人非商业使用免费；商用需购买授权
- **回退**：未安装时自动使用宋体（排版合规，红头视觉略逊）

## 手动安装（备选）

1. 双击 `fonts/方正小标宋简体.ttf`
2. 点击"安装"按钮
3. 重启 Word/WPS
